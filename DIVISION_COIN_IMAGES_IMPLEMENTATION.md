# Division-Specific Coin Images Implementation

## Overview

This implementation adds division-specific coin images to the E-Sport Connection platform. Each division (Silver, Gold, Diamond) now displays its own unique coin image instead of using the same default gold coin everywhere. The system defaults to Gold division coins when division information is not available.

## Implementation Details

### 1. Division Service Updates

**File**: `frontend/src/services/divisionService.ts`

Added new method:

```typescript
static getDivisionCoinImage(division: SquadDivision): string {
  switch (division) {
    case SquadDivision.SILVER:
      return "https://res.cloudinary.com/djvjsyzgw/image/upload/v1756565086/Gemini_Generated_Image_rf84kyrf84kyrf84_kf8gkk.png";
    case SquadDivision.GOLD:
      return "https://res.cloudinary.com/djvjsyzgw/image/upload/v1756557908/coin_masl_nzwekq.png";
    case SquadDivision.DIAMOND:
      return "https://res.cloudinary.com/djvjsyzgw/image/upload/v1756557908/coin_masl_nzwekq.png";
    default:
      return "https://res.cloudinary.com/djvjsyzgw/image/upload/v1756557908/coin_masl_nzwekq.png";
  }
}
```

### 2. DivisionCoinImage Component

**File**: `frontend/src/components/DivisionCoinImage.tsx`

Created a reusable component that:

- Automatically displays the correct coin image based on division
- Supports customizable sizes
- Optional glow effects with division-specific colors
- Responsive design with proper alt text

**Features**:

- `division`: Required - The division to display coin for
- `size`: Optional - Coin size in pixels (default: 48)
- `className`: Optional - Additional CSS classes
- `showGlow`: Optional - Whether to show animated glow effect

## Usage Examples

### Basic Usage

```tsx
import DivisionCoinImage from "../components/DivisionCoinImage";
import { SquadDivision } from "../types/division";

<DivisionCoinImage division={SquadDivision.SILVER} size={48} />;
```

### With Glow Effect

```tsx
<DivisionCoinImage division={SquadDivision.DIAMOND} size={64} showGlow={true} />
```

### With Custom Classes

```tsx
<DivisionCoinImage
  division={SquadDivision.GOLD}
  size={32}
  className="opacity-80"
/>
```
